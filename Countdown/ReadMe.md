# Countdown

Write a recursive function `countdown(int n)` that prints every integer from `n` down to `1`, each on its own line, followed by `"Go!"` on the last line.

Your `main` should read a single integer from the user and call `countdown` with it.

---

## Sample Input

```
5
```

## Sample Output

```
5
4
3
2
1
Go!
```

---

## Constraints

- `1 ≤ n ≤ 100`
- The function **must** be recursive — no loops allowed inside `countdown()`
- `"Go!"` must be printed by the recursive function, not in `main`
